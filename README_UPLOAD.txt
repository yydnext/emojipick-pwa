EmojiPick PWA (White theme) — Upload Instructions

You must upload ALL core files together so they match:
- index.html
- app.js
- styles.css
- sw.js
- manifest.json
- manifest.webmanifest
- og.png
- icon-192.png
- icon-512.png

GitHub web upload (no Git Bash needed)
1) Open the repo: https://github.com/yydnext/emojipick-pwa
2) Click: Add file -> Upload files
3) On your PC, open the folder where you extracted this zip.
4) Select ALL files listed above (do not select the folder itself).
5) Drag the selected files into the GitHub upload area.
6) At the bottom, click “Commit changes”.

After upload
- Open your site: https://yydnext.github.io/emojipick-pwa/
- If you still see the wrong page, force-refresh (Ctrl+Shift+R).
- If it still persists, clear site data for yydnext.github.io (cookies + cache) and reload.
