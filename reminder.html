Buy/Check Reminder

This feature is for convenience only. It helps you remember when to buy and when to check results. It does not increase your odds or recommend purchases.

How it should work (v1)

  • Buy reminder: a notification or calendar event on draw days (e.g., “Buy ticket before cutoff”).
  • Check reminder: a follow-up reminder after the draw time (e.g., “Check winning numbers”).

Responsible play (required)

Lottery games involve risk. Please play responsibly, follow local laws, and seek help if gambling becomes a problem. (US help line: 1-800-522-4700)

Note: In the current web demo we do not request notification permissions to keep onboarding simple. Calendar/notification integration will be added once the core UX is stable.
